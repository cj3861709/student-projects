# 基于多维特征指标的心脏病智能预测

本科课程大作业：数据挖掘

## 项目简介

基于 Kaggle 公开的心脏病数据集，构建并对比逻辑回归、K 近邻、决策树、随机森林四种机器学习模型，通过特征工程与模型优化实现高精度的心脏病风险预测。

## 技术栈

- Python
- scikit-learn（逻辑回归 / KNN / 决策树 / 随机森林）
- Pandas, NumPy
- Matplotlib

## 模型对比

| 模型 | 准确率 | 召回率 | AUC | 漏诊率 |
|------|--------|--------|-----|--------|
| 逻辑回归 | 89.13% | 92.45% | 0.9521 | 7.55% |
| K 近邻 | 86.96% | 88.68% | 0.9387 | 11.32% |
| 决策树 | 90.22% | 94.34% | 0.9648 | 5.66% |
| **随机森林** | **95.65%** | **98.10%** | **0.9845** | **1.90%** |

**最佳模型**：随机森林 — 高召回率（漏诊率仅 1.90%），针对医疗场景"早发现、不漏诊"优化。

## 核心思路

1. 特征工程：临床多维指标标准化、相关性分析、特征筛选
2. 模型集成对比：四种经典分类算法系统对比与调优
3. 优化策略：类别加权平衡、阈值下采样、交叉验证
4. 重点优化召回率，降低漏诊风险
