# 课程项目集 - 陈锦 (CJ3861709)

本仓库包含我大学期间完成的三个课程项目，涵盖**大数据实时处理**、**Android 移动开发**和**数据挖掘与机器学习**三个方向。

---

## 📂 项目列表

### 1. [基于 Spark 的股票市场实时行情分析与预测系统](./stock-market-analysis/)

使用 Apache Spark Streaming + Kafka + XGBoost 构建的实时股票行情分析预测系统，覆盖数据采集 → 流式处理 → 特征计算 → 模型预测 → Web 可视化的完整数据链路。

### 2. [基于 Android Studio 个人"数字名片"生成器](./digital-business-card/)

采用 Kotlin + Jetpack Compose + MVVM 架构开发的 Android 名片应用，支持多模板切换、vCard 二维码生成、相机/相册头像选取、一键分享等功能。

### 3. [基于多维特征指标的心脏病智能预测](./heart-disease-prediction/)

基于 Kaggle 心脏病数据集，使用 scikit-learn 构建并对比逻辑回归、KNN、决策树、随机森林四种分类模型，实现高精度的心脏病风险预测。
